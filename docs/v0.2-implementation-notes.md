## Retrieval Behavior

`prepare.py` is the skill-facing entrypoint. It emits JSON only and enriches database rows with source metadata from `sources.v0.2.yaml`.

`prepare.py --stats` emits JSON-only source and database statistics for agent workflows.

`search_v0_2.py` is for human inspection. It uses the same preparation logic but prints readable output.

Default result ordering prefers higher-priority sources, so broad art/culture media should not dominate evidence-style answers.
